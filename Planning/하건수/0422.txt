- 아이디어 회의
🎯 실시간 오렌지 당도 예측기 (비접촉 · 멀티타겟 지원 · 자동 인식)
📱 카메라에 오렌지를 비추기만 하면,
🤖 화면에 자동으로 다수의 오렌지를 감지해서,
📊 각 오렌지에 당도(Brix)를 실시간으로 표시하는 구조

✅ 기능 정리

기능	설명
📸 실시간 카메라 피드	비추기만 해도 자동 분석 (촬영 없음)
🟠 오렌지 감지 (여러 개)	이미지 내 오렌지를 여러 개 인식 (Object Detection)
🟥 각 오렌지에 RGB 추출	감지된 각 오렌지에서 ROI 추출 후 RGB 평균 계산
🔢 Brix 예측 및 표시	모델로 예측 후, 오렌지 위에 12.5°Bx 식으로 실시간 출력
🌀 지속 반복	매 프레임마다 자동 분석 (OpenCV or WebRTC + ML inference)
