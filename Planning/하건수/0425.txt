스마트폰 활용 수박 당도 예측 AI 연구 조사 보고서

✅ 1. 개요 및 배경
전통적 수박 선별법(외관/소리)은 주관적이고 편차가 큼

AI와 스마트폰 기술로 비파괴적 예측 시도 증가

주요 분석 방식: 외관 이미지 분석 + 타격음 분석

✅ 2. 수박 외관 기반 당도 예측 연구
줄무늬, 색상, 대칭성 등 시각 정보로 당도 추정 가능

CNN 기반 이미지 분석으로 품질 평가 성능 향상

Ozdemir(2023): ChatGPT로 고른 수박이 실제 시식 평가에서 1위

Chawgien(2021): 외관 + 센서 융합 → 정확도 개선

롯데마트 사례: 8방향 사진 + 중량 → 밀도 계산해 당도 11브릭스 이상만 판매

✅ 3. 수박 타격음 기반 당도 예측 연구
울림 소리로 숙성도 판단 (맑고 깊은 소리 = 잘 익음)

김기훈(2020): 스펙트럼 분석 + SVM → 고/저당도 분류 정확도 83.2%

Li(2025): 타격음 + HHT/NMF → 3단계 숙도 분류 정확도 최대 92.2%

스마트폰 마이크로도 측정 가능하나, 잡음/타격 위치 등 변수 존재

딥러닝, 필터링, 평균화로 실용성 보완 가능

✅ 4. AI 모델링 전략 및 실시간 처리
외관: CNN 기반 (MobileNet, EfficientNet 등)

소리: 1D CNN / 스펙트럼 기반 분류기 (SVM 등)

실시간 예측 가능 (수 초 내 처리)

멀티모달(외관+소리+무게) 접근 → 정확도 향상

모델 경량화 필요 시 지식 증류/양자화 활용
