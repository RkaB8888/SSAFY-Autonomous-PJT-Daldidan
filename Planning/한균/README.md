### 1. **내 말버릇 분석기**

### 아이디어

- 내가 자주 쓰는 말버릇, 접속사, 어미, 감탄사 등을 분석하는 웹 앱

- 자주 쓰는 단어 TOP10 시각화
- 긍정/부정 어휘 비율 분석

### 기술 스택

- 프론트엔드: React
- 백엔드: Python (Flask or FastAPI)
- NLP 라이브러리: KoNLPy, PyKoSpacing, soynlp 등

### 🍊 SweetOr – 오렌지 실시간 당도 예측 앱 (서버 추론 기반 MVP)

#### 📌 개요  
스마트폰으로 오렌지를 비추기만 하면 1초마다 당도 예측 결과가 실시간으로 표시되는 PWA 웹앱 서비스

#### 🔍 핵심 흐름
- getUserMedia로 카메라 뷰 활성화
- 매 1초마다 `<canvas>`로 프레임 캡처
- 흐림 감지(OpenCV.js) → 서버 전송 → PyTorch 추론 → UI 렌더링
- 흐림 시 자동 경고: “카메라를 고정해주세요!”

#### 🧠 기술 개요
| 항목 | 내용 |
|------|------|
| 추론 방식 | FastAPI 서버 + PyTorch 모델 |
| 흐림 감지 | OpenCV.js + Laplacian Variance |
| 데이터셋 | 직접 촬영 1,000개 + 굴절 당도계 |
| 출력 | 점수(`12.1°Bx`) + 문장(`상급 오렌지입니다! 🍊`)

#### ⚖️ 서버 추론 vs On-device 비교
| 항목 | 서버 추론 | On-device |
|------|-------------|------------|
| 속도 | 0.5~1초 | 100~200ms |
| 오프라인 | ❌ | ✅ |
| 유지보수 | 쉬움 | 어려움 |
| UX | 충분히 빠름 | 최고 수준 UX

#### 🧪 개발 전략
1. 서버 추론 MVP부터 완성 (흐림 감지 포함)  
2. 여유 시 TensorFlow.js 등으로 On-device 추론 전환 고려

const video = document.querySelector("video");

});

const canvas = document.createElement("canvas");
const ctx = canvas.getContext("2d");
canvas.width = 224;
canvas.height = 224;

setInterval(() => {
  ctx.drawImage(video, 0, 0, canvas.width, canvas.height);
  const dataURL = canvas.toDataURL("image/jpeg", 0.5);
  sendImageToBackend(dataURL);
}, 1000);

const socket = new WebSocket("ws://localhost:8000/ws");
function sendImageToBackend(dataURL) {
  const base64 = dataURL.split(',')[1];
  socket.send(base64);
}
socket.onmessage = (e) => {
  const result = JSON.parse(e.data);
  console.log("예측 결과:", result);
};

## 후지 사과 외관 특징 분석 및 당도 예측 방향성 수립

---

### 🍎 달디단 – 후지 사과 외관 특징 분석

#### 🎯 문제 인식 및 변화
- 후지 사과는 붉은색(a\* 값)이 높을수록 당도(SSC)와 양의 상관관계(r=0.626)를 가짐.
- 그러나 붉은 정도만으로 SSC를 완벽히 예측할 수는 없음.
- 크기(무게, 직경), 모양(종횡비), 결점(밀병 제외)은 SSC와 큰 연관이 없음.
- ➡️ **단일 색상 정보가 아닌 색상 + 질감 조합 기반 예측 모델** 필요.

---

### 🛠 새로운 방향: 색상 및 질감 조합 기반 머신 비전 모델

#### 📋 구체적인 접근 방법
- 단순 색상 정보(a\*, b\* 값)만으로는 중간 수준(Rp² ≈ 0.6) 정확도에 그침.
- 다양한 색 공간(RGB, YCbCr 등) 변환 + 질감 특징을 추가하여 예측 성능 개선 가능.
- 고급 모델(ANN-CA 등) 사용 시 높은 정확도(R² > 0.92) 달성 가능성 확인.
- 조명, 품종 차이 등 외부 환경 변화에 민감하므로 데이터 전처리 및 표준화 필요.

#### 📈 목표
- 색상(a\*) 중심 Feature 추출을 기본으로, 다양한 시각적 Feature 조합
- 초기에는 규칙 기반 점수화 → 이후 Calibration 데이터셋으로 보정
- 작은 데이터 기반 실질적 당도 선호 예측 시스템 구축

---

### 🔍 참고 및 아이디어 기반 자료
- 붉은색(a\*) 값: 당도와 가장 높은 상관관계
- 전체 색상(b\*, hue angle 등): a\*와 조합해 당도 예측 정밀도 상승
- 광택 및 질감 정보: 신선도 및 외관 품질 지표로 활용 가능
- 크기, 모양: 당도 예측에 큰 기여 없음
- 밀병(watercore): 인지되는 단맛은 상승할 수 있으나, SSC 일관성은 낮음

- **수정**: 색상(a\* 중심) + 질감 특징 조합 기반 머신 비전 모델로 방향 전환
